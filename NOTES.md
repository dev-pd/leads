# NOTES — attribution

I drove this project: I planned it, decomposed it into phases, made every
architecture decision, reviewed all output, ran the stack to verify, and wrote
or hand-edited the load-bearing parts. A coding agent (Claude Code, Opus 4.8)
generated code under my direction. This file records origin per area; see
[`docs/AGENT_USAGE.md`](docs/AGENT_USAGE.md) for the writeup and
[`docs/PROMPTS.md`](docs/PROMPTS.md) for representative prompts.

## Legend
- **Author-driven** — I made the decision/design; code emitted under close
  direction and reviewed, edited where needed.
- **Author-written** — written or substantially rewritten by hand.
- **Agent** — generated by the agent, reviewed (and where needed fixed) by me.
